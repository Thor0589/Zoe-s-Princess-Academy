# Zoe's Princess Academy Voice Script Manifest

This package plans pre-recorded MP3 coverage for the current single-file app in `index.html`. It does not replace the existing `speechSynthesis` flow. Curriculum-specific prompts should continue to come from existing `data.audio`, `data.phonemes`, `data.item`, and `data.text` values unless a later audio pass generates per-item files.
